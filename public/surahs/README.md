# Surah audio

Drop an MP3 for each surah here, named by its **id** (from `lib/surahs.js`):

```
public/surahs/al-fatihah.mp3
public/surahs/yasin.mp3
public/surahs/ar-rahman.mp3
public/surahs/al-waqiah.mp3
public/surahs/al-mulk.mp3
public/surahs/al-kahf.mp3
public/surahs/al-ikhlas.mp3
public/surahs/al-falaq.mp3
public/surahs/an-nas.mp3
```

A surah with no file yet just shows "audio not added". To add more surahs,
add an entry to `lib/surahs.js` and drop the matching `<id>.mp3` here.
